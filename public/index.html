Canal público de instalação da base financeira privada, self-hosted e AGPL-3.0.

Beta Pública

Recomendada

Instale a versão beta mais recente. O canal é atualizado a cada nova release beta.

curl -fsSLo /tmp/contabase-install.sh https://get-contabase.pages.dev/beta/install.sh && bash /tmp/contabase-install.sh

Estável

Em breve

Disponível após a primeira release estável.

Versão Específica

Manual

Fixe uma versão específica com a variável CONTABASE_VERSION:

curl -fsSLo /tmp/contabase-install.sh https://get-contabase.pages.dev/beta/install.sh && CONTABASE_VERSION=v0.1.0-beta.3 bash /tmp/contabase-install.sh
